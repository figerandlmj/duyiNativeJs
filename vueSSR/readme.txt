SSR  (server side render) 服务器端渲染

服务器端发展史
    1994 网景 浏览器
    万维网（www）
    文档的内容,基于html
    所有的页面都是html,没有js,没有交互
    浏览器不支持运算，所有运算都在服务器上
    所有页面都是静态的
    1994 php (personal home page) 模板引擎  服务器端渲染
    不用重复造轮子
    1995 js 实现客户端计算
    1999 html4.0 微软 ActiveX，XMLHttpRequest 让web页面进入2.0
    2006 ajax 进入规范
    2008 浏览器大战 chrome v8
    2009 angularJS backbone.js
    2011 react
    2014 vue
    前后端分离
    前端： 拿数据 写页面逻辑
    后端： 写数据 搭建服务
    服务器端渲染 =》 前后端分离 =》 nodeJs 服务器端 （全站）

优点
    更好的SEO
    首屏加载速度更快
    牺牲服务器计算时间，换取一次数据请求
