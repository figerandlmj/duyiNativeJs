1.块级作用域 let const

全局作用域  函数作用域

let 块级作用域
	不存在变量提升  先声明后使用
	暂时性死区  声明之前使用就报错
	不允许重复声明
const 常量 不可以改变的

变量的解构赋值  模式匹配
	数组的解构赋值
	let [a,b,c] = [1,2,3];
	let [a,[b,[c]]] = [1,[2,[3]]];

	解构不成功 不完全解构 
	支持默认值 惰性赋值 找不到值时走默认值

	对象解构赋值 没有顺序
	let {foo,bar} = {foo:123,bar:234} -> {foo:foo,bar:bar} = {foo:123,bar:234}
	let {foo:bar} = {foo:123}  bar->123  foo -> error
